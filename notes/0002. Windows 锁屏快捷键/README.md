## 1. 📒 `win L`

- **win + L**
  - 这是锁定屏幕的最快方式。
  - 按下这个组合键后，你的电脑会立即转到锁屏界面。
  - 如果有设置开机密码，则要求输入密码或其他验证信息才能重新进入系统。
