## 3. 🤔 LICEcap 是什么？

### 3.1. LICEcap 简介

LICEcap 是一个非常流行且易于使用的免费工具，可以录制屏幕并保存为 GIF 格式。

特点：

- 免费
- macos、windows 均支持
- 可配置录制参数

目前 `24.10` LICEcap 实际测试下来是可用的，使用步骤也比较简单，可以在官方文档中点击 view a demo 来快速了解这玩意儿应该如何使用。

![img](https://cdn.jsdelivr.net/gh/tnotesjs/imgs@main/2024-10-14-10-29-36.png)

### 3.2. LICEcap 的安装和使用步骤

#### 安装

访问 [LICEcap 官方网站][1] 下载 LICEcap。

![img](https://cdn.jsdelivr.net/gh/tnotesjs/imgs@main/2024-11-30-12-00-23.png)

#### 使用步骤

1. 打开 LICEcap
2. 先选择你准备将录制好的 gif 图存放在什么位置，文件名叫什么
3. 根据实际需要配置录制参数 - 也可以简单些，直接使用默认的即可，不需要修改参数配置的值
4. 将录制的框挪动到你需要录制的位置，点击开始录制，在录制完毕后点击 stop 结束即可

![img](https://cdn.jsdelivr.net/gh/tnotesjs/imgs@main/2024-11-30-12-07-42.png)
