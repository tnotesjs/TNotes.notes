## 2. 🫧 评价

该笔记对 ScreenBrush 这款 macOS 上的屏幕画笔工具做了一个简单的介绍。

在 macOS 中录制一些在线视频教程的时候，这个工具还蛮合适的，可以在屏幕上进行一些标注和绘图，方便讲解内容。

- 功能不少，但使用门槛也不低
- plus 版价格是 ¥38 一次性买断，如果能熟练使用 ScreenBrush 并且使用频率较高的话，还是比较推荐的

## 3. 🤔 ScreenBrush 是什么？

### 3.1. ScreenBrush 简介

ScreenBrush 是一款 macOS 上的屏幕画笔 🖌 工具，用于在屏幕上在线绘制一些图案，比如：

- 线段（可以带剪头也可以不带剪头）
- 矩形框
- 圆
- ……

### 3.2. 适用人群

在 macOS 上，需要在屏幕上添加临时笔记的情况下需要用到，比如录课、在线讲课……

### 6.3. 功能 - 进入绘图模式，但是不显示工具栏

- 官方术语：`Show Without Tollbar`，表示打开 ScreenBrush 显示画笔，但是不要显示工具栏
- 默认快捷方式：`ctrl tab`
- 显示画笔，意味着进入绘图模式
- 如果想要退出绘图模式，可以按下 `esc` 或者点击鼠标右键

### 6.4. 功能 - 使用曲线画笔

- 官方术语：`Brush & Quick Arrow Tool`，表示使用曲线画笔
- 默认快捷方式：`cmd 1`
- 重复按下 `cmd 1` 可以切换曲线是否带有剪头

### 6.5. 功能 - 使用直线画笔

- 官方术语：`Arrow & Line Tool`，表示使用直线画笔
- 默认快捷方式：`cmd 2`
- 重复按下 `cmd 2` 可以切换曲线是否带有剪头

### 6.6. 功能 - 使用矩形画笔、椭圆形画笔

- 官方术语：`Rectangle & Ellipse Tool`，表示使用矩形画笔、椭圆形画笔
- 默认快捷方式：`cmd 3`
- 重复按下 `cmd 3` 可以切换矩形还是椭圆形

### 6.7. 功能 - 使用文本输入框

- 官方术语：`Text Tool`，表示使用文本输入框
- 默认快捷方式：`cmd 4`
- 鼠标左键：在需要编辑文本的地方单击鼠标左键，即可进入文本输入模式
- 回车键：文本内容编辑完之后，按下回车表示完成本次编辑
- 修改文本大小：按下鼠标左键不松手拖动，可以调节文本字体的大小

### 6.8. 功能 - 撤销

- 官方术语：`Undo`，表示撤销
- 默认快捷方式：`cmd Z`，这个没啥好讲的，很符合直觉，大多编辑器的撤销快捷方式都是 `cmd Z`

### 6.9. 功能 - 局部高亮

- 官方术语：`Flashlight`，表示局部高亮功能
- 默认快捷方式：\`（数字 1 左边的那个键） 或者 `shift`

### 6.10. 功能 - 局部高亮缩放

- 在局部高亮的状态下，按下 `cmd` 键的同时，滚动鼠标滚轮

### 6.11. 功能 - 清屏

- 官方术语：`Clear Screen`，表示清空屏幕上的所有笔记
- 默认快捷方式：`cmd backspace`
- 除了使用默认的快捷方式之外，还可以先 `cmd A` 选中所有笔记，然后按下 `backspace` 删除所有笔记

### 6.12. 功能 - 拖动指定笔记

1. 选中指定的笔记内容：在绘图模式下，按下 `cmd` 键加鼠标左键
2. 可以拖拽，修改该笔记的位置

### 6.13. 功能 - 删除指定笔记

1. 按下 `cmd` 键，并在指定笔记上双击鼠标左键进入编辑模式
2. 按下 `backspace` 键，删除该笔记
